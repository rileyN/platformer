501  /* In s7, (define* (name . args) body) or (define name (lambda* args body))
502  * define a function that takes optional (keyword) named arguments.
503  * The "args" is a list that can contain either names (normal arguments),
504  * or lists of the form (name default-value), in any order. When called,
505  * the names are bound to their default values (or #f), then the function's
506  * current arglist is scanned. Any name that occurs as a keyword (":name")
507  * precedes that argument's new value. Otherwise, as values occur, they
508  * are plugged into the environment based on their position in the arglist
509  * (as normal for a function). So,
510  *
511  * (define* (hi a (b 32) (c "hi")) (list a b c))
512  * (hi 1) -> '(1 32 "hi")
513  * (hi :b 2 :a 3) -> '(3 2 "hi")

515  *
516  * :rest causes its argument to be bound to the rest of the arguments at that point.
517  *
518  * The C connection to this takes the function name, the C function to call, the argument
519  * list as written in Scheme, and the documentation string. s7 makes sure the arguments
520  * are ordered correctly and have the specified defaults before calling the C function.
521  * s7_define_function_star(sc, "a-func", a_func, "arg1 (arg2 32)", "an example of C define*");
522  * Now (a-func :arg1 2) calls the C function a_func(2, 32). See the example program in s7.html.
523  *
524  * In s7 Scheme, define* can be used just for its optional arguments feature, but that is
525  * included in s7_define_function. s7_define_function_star implements keyword arguments
