Detailed Description

Defines a scene of entities and components.

A component multi means that an entity can have multiple of these components. The return type of the getter for these multi components can be iterated over.

Use has_* to check if a entity contains a component. Use get_* to get the component of an entity. Use add_* to add a component to an entity. Use set_* to set the component of an entity.
